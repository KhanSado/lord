# Welcome to Lord UI !
Lord is library with UI components using jetpack compose
⚠ **Under Construction**

## Setup

Add it in your root build.gradle at the end of repositories:
```css
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
Add the dependency
```css
dependencies {
	 implementation 'com.github.karontyy:lord:last_version'
}
```
